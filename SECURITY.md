# Security Policy

## Supported Versions

This project is alpha. Security fixes are handled on the main branch until release channels are formalized.

## Reporting

Please do not open public issues containing credentials, private ChatGPT content, account identifiers, tokens, or sensitive browser state. Open a minimal issue that describes the class of problem and offer to share details privately.

## Boundaries

This SDK is for user-directed workflows in a visible ChatGPT web session. It must not be used to bypass authentication, evade product controls, call private ChatGPT endpoints, or collect data from sessions the user did not intentionally expose to the running agent.
